# COSSブログ AIヒューマナイザー指示書

ヨガ記事のリライトから抽出したルール。新規記事生成時のAIへの指示としてそのまま使える。

---

## NG表現リスト（日本語版）

### 文体・語尾

| NG | 代替 |
|----|------|
| 「安心してください」 | 使わない。直接本題に入る |
| 「〜ですよね？」の多用 | 1記事に1〜2回まで |
| 「さあ、〜しましょう」 | 使わない。具体的なアクションか、素直な締めに |
| 「一緒に〜しましょう」「見ていきましょう」 | 使わない。そのまま始める |
| 「〜はずです」（根拠のない断言） | 「〜ことが多い」「〜傾向があります」 |
| 「〜からこそ」の多用 | 1記事に1回まで |
| 「内側と外側の両方から」 | 具体的に何と何を組み合わせるか書く |
| 「多角的に働きかけてくれます」 | 何がどう変わるかを具体的に |

### 記号・構造

| NG | 代替 |
|----|------|
| 「——」（ダッシュ）の多用 | 読点「、」か句点「。」で文を切る |
| 商品紹介の箇条書き「特徴:」セクション | 本文の流れの中に溶け込ませる |
| まとめを箇条書きにする | 1〜2文の実用的なアドバイスで終わる |
| キーワード＋コロンの見出し（「ヨガ 効果のポイント:」） | 使わない |

### CTA（商品紹介）への導入

| NG | 代替 |
|----|------|
| 「そんなときに便利なのが〜」 | ユーザーの具体的な場面を先に描写してからつなぐ |
| 「私たちCOSSでは〜を提供しています」 | 「〜ために開発したのが〜です」など目的から入る |
| 製品特徴の箇条書き | 1〜2文で「何ができるか」「いくらか」だけ書く |

## やること

### 構造のバラツキを出す

AI文章は各段落・各セクションが均等な長さになりやすい。
意図的にバラつかせる：

- 短い段落（1〜2文）と長い段落（4〜5文）を混ぜる
- 全セクションを同じ形式にしない
- 完璧なタイムライン（1週間/1ヶ月/3ヶ月/半年/1年）を使うときは、各段階の説明量を変える

### 具体性を上げる

| 曖昧 | 具体的 |
|------|--------|
| 「効果が期待できます」 | 「ウエスト周りが引き締まってくる」 |
| 「変化を実感できる時期です」 | 「周囲から『痩せた？』と聞かれるのはこの頃」 |
| 「肌の調子が良くなります」 | 「ファンデーションのノリが変わる」 |

### 締めのパターン

NG：「〜あなたの明日がもっと輝くはずです」「さあ、一歩を踏み出しましょう」

OK：
- 「〜なら、○○を試してみるといいかもしれません」（具体的アドバイス）
- 「〜が大事です。それだけです」（シンプルに切る）
- 「〜については別記事で詳しく書きます」（次の誘導）

## 運動後ケアへの導線ルール（COSS専用）

このブログは「運動する人」に「スキンケア商品」を自然に知ってもらうことが目的。

1. **記事本文中で1回**: 運動と肌の関係を科学的に書く（血行促進→肌変化、汗→毛穴詰まりなど）
2. **専用セクションで1回**: ヨガ/筋トレ/ランニング後の具体的なスキンケア手順を書く
3. **商品紹介は手順の自然な流れから**: 「時間がないときのショートカット」として紹介する

商品紹介の基本形：
```
[場面描写（帰り道、朝ヨガ後など）] + [課題（時間がない/荷物が多い）] 
→ 「そういう場面で使いやすいように開発したのが〜」
→ [商品名 + 1文で何ができるか + 価格]
→ [リンク]
```

## 最終チェック（記事投稿前）

- [ ] 「——」が使われていないか
- [ ] 「安心してください」「さあ〜しましょう」がないか
- [ ] まとめが箇条書きになっていないか
- [ ] 商品紹介が唐突に「便利なのが〜」で始まっていないか
- [ ] 結びが「輝く未来」系の抽象的な応援で終わっていないか
- [ ] 各段落の長さに多少のバラツキがあるか
