## ❌ AI文体禁止リスト（最重要）

以下の表現・パターンは**絶対に使用しないこと**。これらはAIが書いたと一目でわかる文体です。

### 禁止フレーズ

| 禁止 | 理由 |
|------|------|
| 「安心してください」 | AIの定番の読者なだめフレーズ |
| 「さあ、〜しましょう」「さあ、〜へ！」 | AIの締め・呼びかけの定番 |
| 「一緒に〜しましょう」「一緒に〜しよう」 | 仲間感の演出がAIっぽい |
| 「〜を見ていきましょう」「〜を解説していきます」 | 読む前に予告するAIの癖 |
| 「そんなときに便利なのが〜」 | AI商品紹介の定番導入 |
| 「〜はずです」（根拠のない断言） | AIの空虚な確信 |
| 「内側と外側の両方からケアする」 | バランス感が作られすぎ |
| 「多角的に働きかけてくれます」「多面的に」 | 曖昧なAI修飾語 |
| 「〜ではありません。〜です」（否定→肯定） | AIの定番コントラスト構文 |

### 禁止記号・形式

| 禁止 | 代替 |
|------|------|
| 「——」（ダッシュ）の多用 | 読点「、」か句点「。」で切る |
| まとめを箇条書き（✓や・）にする | 1〜2文の文章で書く |
| 「キーワード 効果のポイント:」型の見出し | 使わない |
| 商品紹介を箇条書き「特徴:」で書く | 本文に自然に溶け込ませる |

### 禁止する締めのパターン

- 「〜あなたの毎日がもっと輝くはずです」
- 「まず一歩を踏み出してみましょう」
- 「新しい自分に出会えるはずです」
- 「今日から始めてみてください」（単体で使う場合）

**代わりに使うもの：** 具体的なアクションアドバイスで締める
- 例：「週2〜3回から始めると続きやすいです」
- 例：「まず1週間試してみて、肌の変化を観察してみてください」
- 例：「〜が気になるなら、まず○○を見直すといいかもしれません」

## 記事の構成要件

### 1. 導入部（200〜250文字）

**書き方：**

- {キーワード}を検索した人が何を知りたいかを1〜2文で具体的に書く
- すぐに本題に入る
- 「この記事で分かること:」のリストは**不要**（読む前に予告しない）

**NG例：**
> 「{キーワード}について気になっていませんか？この記事では〜を解説していきます。この記事で分かること：✓〜 ✓〜 ✓〜」

**OK例：**
> 「{キーワード}の効果は、正直なところ時期によって全然違います。最初に変わるのは（略）」

**トーン：**
- 落ち着いた文章。仲間を演じない
- 「〜ですよね？」は1記事に2回まで
- 「〜かもしれません」「〜ことが多い」を活用する（断定しすぎない）

---

### 2. 基礎知識・メリット（400〜500文字）

**{カテゴリ}が「運動・フィットネス」の場合：**

- 運動の効果を具体的に書く（「代謝が上がる」だけでなく、何がどう変わるか）
- 必ず「血行促進→肌への影響」に触れる
- 「汗をかいた後のスキンケアを怠ると何が起きるか」を1文入れる

**{カテゴリ}が「スキンケア基本」「肌悩み」の場合：**

- スキンケアの基本的な効果
- 運動との関連性（汗・毛穴・血行）
- 「運動後は特に注意が必要」という流れを作る

**{カテゴリ}が「年代別」の場合：**

- その年代特有の肌の変化を具体的に
- 運動習慣との関連性
- スキンケアの重要性

**共通：**
- 具体的な数字・事実を入れる
- よくある誤解を1つ解消する
- 段落の長さをバラつかせる（全部同じ長さにしない）

---

### 3. 核心コンテンツ（800〜1,000文字）

**{カテゴリ}別の内容：**

**運動・フィットネス：**
- 具体的なメニューや方法
- やり方・回数・ポイントを実用的に
- 初心者向けの調整方法

**スキンケア基本：**
- 正しい手順
- よくある間違い（NG行動）
- 時短のコツ

**肌悩み：**
- 悩みの原因（具体的に）
- 改善方法
- やってはいけないこと

**必須：**
- 実践できる形で書く
- 具体的な手順や数字を含める
- 各段落の長さに意図的にバラツキを出す（均等にしない）

---

### 4. 運動後のケアセクション ★必須（300〜400文字）

**H2見出し例：**
- 「{キーワード}後のスキンケア」
- 「運動後の肌を守るケア」
- 「汗をかいた後に必要なこと」

**内容（この順序で書く）：**

1. **なぜ運動後の肌が特殊な状態なのか**
   - 体温上昇で毛穴が開いている
   - 汗で水分・皮脂が失われている
   - 乾燥しやすい状態

2. **基本的なケアの手順**
   - ぬるま湯での洗顔（ゴシゴシNG）
   - すぐに保湿

3. **商品の紹介（この書き方で）**

   ```
   [場面描写：スタジオ帰り、朝ヨガ後、ジム後など具体的に]
   [課題：化粧水から始めると時間がかかる、荷物になる等]
   そういう場面で使いやすいように開発したのが、COSS THE GEL（コスザゲル）です。
   [1文で特徴：乾いた肌にそのままつけられる1本完結のオールインワンゲル]
   ¥1,980（約1ヶ月分）。
   ```

**禁止：**
- 「そんなときに便利なのが〜」での導入
- 「COSS THE GELの特徴:」箇条書き
- 「私たちCOSSでは〜を提供しています」

---

### 5. よくある質問（300〜400文字）

**FAQ 2〜3問**

必ず含める質問：
- {キーワード}の効果はいつから？
- {キーワード}特有の疑問を1問

**書き方：**
- 回答は2〜4文で具体的に
- 「〜はずです」ではなく「〜が多い」「〜の傾向がある」

---

### 6. まとめ（150〜200文字）

**形式：**
- 箇条書き・✓リスト**不要**
- 2〜3文の文章で書く
- 最後は具体的なアクションアドバイスで締める

**NG：**
```
**{キーワード}のポイント:**
✓ 項目1
✓ 項目2
✓ 項目3
さあ、新しい自分に会いに行こう！
```

**OK例：**
```
{キーワード}の効果が出るまでの目安をまとめると、（略）。
「続けているけど変わらない」という場合は、頻度を週2〜3回に増やすか、
運動後のスキンケアを見直すと変わることが多いです。
```

- 自然な文脈で使用

### 見出し構造

- H2: 5〜6個
- H3: 各H2に1〜3個

## 文体・品質要件

### 文章のルール

- ですます調
- 1文は50文字以内を目安（たまに長くなっても可）
- 1段落2〜4文
- 各段落・セクションの長さを均等にしない

### 禁止事項（既存）

- 医学的根拠のない断定
- 極端なダイエット法の推奨
- 他社商品の批判・比較
- 「絶対」「必ず」の多用
- 「簡単に痩せる」「すぐに効果が出る」「これだけでOK」
